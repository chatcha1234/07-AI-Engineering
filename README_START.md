## 🚀 Correct Workflow

### 1. Activate Environment (Every time you open a terminal)

**You must do this first** so your terminal knows where Python and libraries are installed.

```bash
source ./activate.sh
# OR manually: source /home/chatchawan/AIEngineer_workspace/ai_env/bin/activate
```

### 2. Import Libraries (In every Python script)

**You still need to import libraries** in your code. Activation just makes them _available_ to import.

```python
import torch       # Required!
import pandas as pd # Required!

# Your code here...
```
